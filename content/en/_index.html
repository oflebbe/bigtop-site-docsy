{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Bigtop is an Apache Foundation project for Infrastructure Engineers and Data Scientists looking for comprehensive packaging, testing, and configuration of the leading open source big data components. Bigtop supports a wide range of components/projects, including, but not limited to, Hadoop, HBase and Spark. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fab" title="Packaging" %}} Bigtop packages Hadoop RPMs and DEBs, so that you can manage and maintain your Hadoop cluster. {{% /blocks/feature %}} {{% blocks/feature icon="fab" title="Smoke Testing" %}} Bigtop provides an integrated smoke testing framework, alongside a suite of over 50 test files. {{% /blocks/feature %}} {{% blocks/feature icon="fab" title="Virtualization" %}} Bigtop provides vagrant recipes, raw images, and (work-in-progress) docker recipes for deploying Hadoop from zero. {{% /blocks/feature %}} Bigtop support many Operating Systems, including Debian, Ubuntu, CentOS, Fedora, openSUSE and many others. Bigtop includes tools and a framework for testing at various levels (packaging, platform, runtime, etc.) for both initial deployments as well as upgrade scenarios for the entire data platform, not just the individual components. {{< /blocks/section >}} {{< blocks/section >}} {{% blocks/feature icon="fab"%}} {{% /blocks/feature %}} {{% blocks/feature title="Download latest release of Bigtop" %}} [Latest Release](http://bigtop.apache.org/download.html#releases) {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}} {{% blocks/feature icon="fab fa-app-store-ios" title="Download **from AppStore**" %}} Get the Goldydocs app! {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/google/docsy-example" %}} We do a [Pull Request](https://github.com/google/docsy-example/pulls) contributions workflow on **GitHub**.
